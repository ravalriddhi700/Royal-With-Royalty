#  Royal With Royalty

Royal With Royalty is my BCA project developed using the Django framework. It is an online jewellery selling website where users can explore different jewellery collections, create an account, add products to the cart, and place orders. The project also includes separate Admin and Staff panels to manage the website efficiently.

---

## About the Project

The main objective of this project is to provide an easy-to-use online platform for buying jewellery. Customers can browse products, view product details, and purchase jewellery from the website. Administrators and staff can manage products, categories, orders, and users through their respective dashboards.

## How to Run the Project

1. Clone this repository

```bash
git clone https://github.com/ravalriddhi700/Royal-With-Royalty.git
```

2. Open the project folder

```bash
cd Royal-With-Royalty
```

3. Install Django

```bash
pip install django
```

4. Start the development server

```bash
python manage.py runserver
```

5. Open your browser and visit

```
http://127.0.0.1:8000/
```
